TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: Python still not found after installation
   → Close and reopen PowerShell
   → Or restart your computer
   → Check PATH environment variable

Issue: "winget not found"
   → Update Windows to latest version
   → Or use manual download links provided above

Issue: Permission errors
   → Run PowerShell as Administrator
   → Right-click PowerShell → "Run as Administrator"

Issue: CAPTCHA appears during login
   → Amazon detected automation
   → Edit config/settings.py: Set headless: False
   → Manually solve CAPTCHA when it appears
