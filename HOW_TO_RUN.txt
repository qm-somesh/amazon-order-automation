STEP-BY-STEP SETUP & RUN GUIDE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Check Prerequisites
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Run in PowerShell:
   python --version        # Should show 3.9 or higher
   node --version          # Should show 16 or higher
   wkhtmltopdf --version   # Check if installed


STEP 2: Run Setup Script (Automated Installation)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Run in PowerShell:
   cd "d:\Info\Learn to Evolve team\AI\amazon-order-automation"
   .\scripts\setup.bat

This will:
   ✅ Create Python virtual environment
   ✅ Install all Python dependencies
   ✅ Install Stagehand AI via npm
   ✅ Create output directories
   ✅ Copy .env.example to .env


STEP 3: Configure Your Credentials
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Edit the .env file (it's already open in your editor!)

Replace these values:
   AMAZON_EMAIL=your_email@example.com      ← YOUR AMAZON EMAIL
   AMAZON_PASSWORD=your_secure_password     ← YOUR AMAZON PASSWORD

Save the file!


STEP 4: Install wkhtmltopdf (If Missing)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

If wkhtmltopdf is not installed:

Option A - Using winget:
   winget install wkhtmltopdf

Option B - Manual Download:
   1. Go to: https://wkhtmltopdf.org/downloads.html
   2. Download Windows installer
   3. Install it
   4. Restart terminal


STEP 5: Run the Automation! 🚀
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Run in PowerShell:
   # Activate virtual environment
   .\venv\Scripts\activate

   # Run the automation
   python src\amazon_automation.py

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 🌐 Browser Opens (Chrome/Chromium)
   → You'll see the browser window open
   → Watch Stagehand AI navigate automatically!

2. 🔐 Automatic Login
   → Navigates to Amazon India
   → Clicks "Sign In" button (using AI!)
   → Enters your email
   → Enters your password
   → Completes login

3. 📊 Extract Orders
   → Navigates to Orders page
   → Uses AI to extract order details
   → No brittle selectors - pure AI magic!

4. 📄 Generate Reports
   → Creates professional PDF report
   → Saves JSON backup
   → Takes screenshots

5. ✅ Complete!
   → Check output/ folder for PDF
   → Check screenshots/ folder for debug images

TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Module not found" error
   → Run: pip install -r requirements.txt

❌ "Stagehand not found" error
   → Run: npm install @browserbasehq/stagehand

❌ Login fails / CAPTCHA appears
   → Amazon detected automation
   → Try running in non-headless mode
   → Edit config/settings.py: Set headless: False
   → You may need to solve CAPTCHA manually

❌ "wkhtmltopdf not found"
   → Install from: https://wkhtmltopdf.org/downloads.html
   → Or use: winget install wkhtmltopdf

❌ Browser doesn't open
   → Check if virtual environment is activated
   → Check internet connection
   → Review logs in amazon_automation.log

❌ No orders extracted
   → Check if you have orders on Amazon
   → Review screenshots/ folder
   → Check amazon_automation.log

CONFIGURATION OPTIONS (config/settings.py):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Customize these settings:

   MAX_ORDERS_TO_SCRAPE = 10          ← Number of orders to fetch
   
   STAGEHAND_CONFIG = {
       'headless': False,              ← Set True to hide browser
       'enable_caching': True,         ← Cache AI responses
       'verbose': 1,                   ← Logging level (0-2)
   }
